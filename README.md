# 2026 하반기 여행 캘린더

`여행계획.md`(또는 `plans/` 폴더의 md 파일)를 읽어 7~12월 달력에 휴가·워케이션·행사를 표시하는 정적 웹앱입니다. 빌드 과정이 없어 폴더 그대로 배포하면 동작합니다.

## 일정 추가/수정 방법

일정 한 줄 형식:

```
- 시작일 ~ 종료일 | 분류 | 장소 | 메모
```

- 하루 일정이면 `~ 종료일`은 생략 가능 (예: `- 2026-08-15 | 행사 | 광복절 |`)
- `분류`는 `해외여행`(파랑) · `워케이션`(초록) · `행사`(주황) 중 하나
- `메모`는 생략 가능

일정을 추가하는 방법은 두 가지입니다.

1. 간단하게: 루트의 `여행계획.md` 파일을 직접 고칩니다.
2. 여러 파일로 관리: `plans/` 폴더에 md 파일을 추가하고 `plans/manifest.json` 목록에 파일명을 넣습니다. (manifest가 있으면 이 방식이 우선 적용되고, 나열된 md들을 모두 합쳐서 표시합니다.)

> 참고: 현재 달력은 디자인 그대로 **2026년 7~12월** 6개월만 표시합니다. 다른 기간도 보이게 하려면 알려주세요.

## Vercel 배포 (GitHub 연동 — 추천)

1. GitHub에 새 저장소를 만들고 이 폴더의 파일을 모두 올립니다.
2. [vercel.com/new](https://vercel.com/new) 에서 그 저장소를 Import 합니다.
3. Framework Preset은 **Other**(정적), Build Command 비움, Output Directory 비움(또는 `.`)으로 두고 Deploy.
4. 배포 완료 후 발급된 URL이 여러분의 여행 캘린더 주소입니다.

이후에는 GitHub에 md 파일만 수정/추가하면 Vercel이 자동으로 재배포합니다.

### 대안: Vercel CLI

```
npm i -g vercel
cd 이폴더
vercel --prod
```

명령 실행 시 안내에 따라 로그인하면 배포됩니다. (업데이트할 때마다 `vercel --prod` 재실행)
